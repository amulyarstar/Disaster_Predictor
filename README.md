# 🌍 TECH SQUAD: AI Disaster Command Center

### Intelligent Early Warning System for Floods & Heatwaves

---

## 🎯 Overview

**TECH SQUAD** is an AI-powered disaster prediction system that analyzes real-time environmental data to forecast **floods** and **heatwaves**.

It enables early warnings so communities can take action before disasters strike.

---

## 🧠 How It Works

1. Collects real-time weather data (OpenWeatherMap API)  
2. Processes inputs using Machine Learning models  
3. Predicts:
   - Flood Risk  
   - Heatwave Risk  
   - Overall Disaster Risk  
4. Applies **temporal logic** to detect continuous extreme conditions  
5. Provides **Explainable AI outputs** with reasoning  

---

## ⚡ Key Features

- 🌦️ Real-time weather data integration  
- 🤖 ML-based predictions (Random Forest)  
- 🌊 Flood risk detection  
- 🔥 Heatwave risk detection  
- 📊 Explainable AI insights  
- ⏱️ Temporal escalation logic  
- 🎛️ Interactive Streamlit dashboard

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/amulyarstar/Disaster_Predictor.git
cd FusionXhackathon
````

### 2. Navigate to project files

```bash
cd code
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run main.py
```
